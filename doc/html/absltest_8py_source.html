560  1. This function uses substring matching, i.e. the matching
561  succeeds if *any* substring of the error message matches *any*
562  regex in the list. This is more convenient for the user than
563  full-string matching.

574

576  regex.
577 
578  Args:
579  actual_str: The string we try to match with the items in regexes.
580  regexes: The regular expressions we want to match against str.
