15 """This package is used to define and parse command line flags.
16 
17 This package defines a *distributed* flag-definition policy: rather than
18 an application having to define all flags in or near main(), each Python
19 module defines flags that are useful to it. When one Python module
20 imports another, it gains access to the other's flags. (This is
21 implemented by having all modules share a common, global registry object
22 containing all the flag information.)
23 
24 Flags are defined through the use of one of the DEFINE_xxx functions.
25 The specific function used determines how the flag is parsed, checked,
26 and optionally type-converted, when it's seen on the command line.
27 """
