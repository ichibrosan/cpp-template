139 // Computes a path suitable as the executable part in CreateProcessA's cmdline.
140 //
141 // The null-terminated executable path for CreateProcessA has to fit into
142 // MAX_PATH, therefore the limit for the executable's path is MAX_PATH - 1
143 // (not including null terminator). This method attempts to convert the input
144 // `path` to a short format to fit it into the MAX_PATH - 1 limit.
145 //
146 // `path` must be either an absolute, normalized, Windows-style path with drive
147 // letter (e.g. "c:\foo\bar.exe", but no "\foo\bar.exe"), or must be just a file
148 // name (e.g. "cmd.exe") that's shorter than MAX_PATH (without null-terminator).
149 // In both cases, `path` must be unquoted.
150 //
151 // If this function succeeds, it returns an empty string (indicating no error),
152 // and sets `result` to the resulting path, which is always quoted, and is
153 // always at most MAX_PATH + 1 long (MAX_PATH - 1 without null terminator, plus
154 // two quotes). If there's any error, this function returns the error message.
155 //
156 // If `path` is at most MAX_PATH - 1 long (not including null terminator), the
157 // result will be that (plus quotes).
158 // Otherwise this method attempts to compute an 8dot3 style short name for
159 // `path`, and if that succeeds and the result is at most MAX_PATH - 1 long (not
160 // including null terminator), then that will be the result (plus quotes).
161 // Otherwise this function fails and returns an error message.
162 wstring AsExecutablePathForCreateProcess(wstring path, wstring* result);
