42  // If present, these flags should either be at the beginning of the command
43  // line, or they should be wrapped in a --wrapper_script_flag=FLAG argument.
44  //
45  // --debug Launch the JVM in remote debugging mode listening
46  // --debug=<port> to the specified port or the port set in the
47  // DEFAULT_JVM_DEBUG_PORT environment variable (e.g.
48  // 'export DEFAULT_JVM_DEBUG_PORT=8000') or else the
49  // default port of 5005. The JVM starts suspended
50  // unless the DEFAULT_JVM_DEBUG_SUSPEND environment
51  // variable is set to 'n'.
52  // --main_advice=<class> Run an alternate main class with the usual main
53  // program and arguments appended as arguments.
54  // --main_advice_classpath=<classpath>
55  // Prepend additional class path entries.
56  // --jvm_flag=<flag> Pass <flag> to the "java" command itself.
57  // <flag> may contain spaces. Can be used multiple
58  // times.
59  // --jvm_flags=<flags> Pass space-separated flags to the "java" command
60  // itself. Can be used multiple times.
61  // --singlejar Start the program from the packed-up deployment
62  // jar rather than from the classpath.
63  // --print_javabin Print the location of java executable binary and
64  // exit.
65  // --classpath_limit=<length>
66  // Specify the maximum classpath length. If the
67  // classpath is shorter, this script passes it to Java
68  // as a command line flag, otherwise it creates a
69  // classpath jar.
70  //
71  // The remainder of the command line is passed to the program.
