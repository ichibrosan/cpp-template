Detailed Description

A simple helper class for creating Jar files. All Jar entries are sorted alphabetically. Allows normalization of Jar entries by setting the timestamp of non-.class files to the DOS epoch. Timestamps of .class files are set to the DOS epoch + 2 seconds (The zip timestamp granularity) Adjusting the timestamp for .class files is necessary since otherwise javac will recompile java files if both the java file and its .class file are present.
